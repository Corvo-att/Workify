Senior React Developer – SaaS Dashboard

Posted 2 hours ago Fixed Price Expert Level 1–3 months 12 applicants

We're looking for an experienced React developer to build and maintain a complex SaaS analytics dashboard for our enterprise clients. You'll work closely with our design team and backend engineers to deliver a polished, performant product.

What You'll Build

  • Interactive data visualization components using Recharts and D3.js
  • Real-time dashboard with WebSocket integration and live updates
  • Role-based access control UI with multiple user permission levels
  • Custom component library following our internal design system
  • Full test coverage with Jest and React Testing Library

Requirements

  • 4+ years of React experience with deep TypeScript proficiency
  • Strong understanding of state management (Redux Toolkit or Zustand)
  • Experience with REST APIs and GraphQL
  • Previous work on data-heavy dashboards or analytics products
  • Ability to communicate clearly and meet deadlines independently

Project Scope

The initial engagement covers the core dashboard MVP — approximately 8 key screens. We expect 4–6 weeks of focused work. If the collaboration goes well, there's strong potential for an ongoing retainer to build additional features.
